### Stuff I made/co-created that I think is cool:

- [Easy RSS](https://github.com/SrKomodo/easy-rss): A Webextension for reading RSS feeds
- [Lucas' Rulers](https://github.com/SrKomodo/lucas-rulers): An HTML5 visualization of Genaille-Lucas rulers
- [Chain Reaction](https://gitlab.com/SrKomodo/chain-reaction): A board game about triggering chain reactions
- [Boolean Formula Solver](https://gitlab.com/SrKomodo/boolean-formula-solver): A website for interacting with boolean formulas, their truth tables, and their circuit diagrams
- [MemeDB](https://gitlab.com/memedb): An absolutely ridiculuous WIP group project involving extending image formats to add tagging for the purpose of meme categorization
- [Mouse Pet](https://github.com/SrKomodo/mouse-pet): A Webextension that adds a little snake following your cursor
- [Ultimate Tetris](https://gitlab.com/SrKomodo/ultimate_tetris): A tetris clone written in Rust
- [Timer](https://gitlab.com/SrKomodo/timer): A pretty simple pomodoro timer, with a very unoriginal name

Besides Github, I also host some of my code on [Gitlab](https://gitlab.com/SrKomodo).
